# PetitAudio
![icon](img/icon.svg)  
A *petit* JS libraries that helps to create interactive audio app  
[GitHub repo](https://github.com/mcbeeringi/petitaudio)  
## features
- method chainable
- audio source array with auto pitch shift for instrument
- filters
	- gain
	- reverb(with IRgen)
	- lowpass(biquad)
	- highpass(biquad)
	- bandpass(biquad)
	- lowshelf(biquad)
	- highself(biquad)
	- peaking(biquad)
	- notch(biquad)
	- allpass(biquad)
	- delay
	- pingpong
